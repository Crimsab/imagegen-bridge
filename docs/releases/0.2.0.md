# Imagegen Bridge 0.2.0

Version 0.2.0 completes the production-resilience model without adding
consumer-specific behavior.

- Independent provider circuit breakers with configurable thresholds,
  three-minute default recovery cooldown, half-open probes, diagnostics,
  metrics, fallback preservation, and honest unknown-outcome handling.
- W3C trace propagation, caller request correlation, provider-attempt spans,
  duration logs, bounded histograms, and effective-provider metrics.
- Deterministic load, stress, spike, soak, and failure-injection tooling plus
  regular and scheduled CI gates.
- OAuth-sterile deployment gateway, mutually exclusive blue/green slots,
  readiness-gated switch, bounded request hold, automatic rollback, and an
  active/passive Docker updater strategy.
- Updated configuration, operations, testing, deployment, and capacity docs.

The release does not include Hermes, Mattermost, Nero Atelier, or other
consumer-specific integrations.
